TimeCheck

Github

A Google chrome extension that modifies the user’s checkout page to find the words “Order Total” and replace them with the user’s number of hours/days worked as payment for the item. This allows the user to clearly see the cost of the purchase in terms of labor required and effectively prevents impulse shopping.
